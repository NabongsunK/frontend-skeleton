## Github 레퍼지토리 복사
* View > Source Control > Clone Repository 선택
* <nohyper>https</nohyper>://github.com/uzoolove/frontend-skeleton.git 입력
* 복사할 적당한 폴더(c://frontend-dev) 선택 후 Select as Repository Destination 선택
  - 폴더 경로에 한글이 들어가지 않도록 주의
* Open 선택
